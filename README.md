## Project Overview

Project Title: Interactive Portfolio Website.
Project Type:Front-end development.
Technologies used: HTML,CSS3,Javascript.

This Project is a modern,responsive and interactive portfolio website.
The website showcase personal details,technical skills,and a contact form with 
real-time validation,providing a smooth and engaging user Experience.

## Project Objectives
- Create an interactive website using Javascript.
- Implement the DOM manipulation and event listener.
- Validate form input using javascript
- Enhance UI/UX with interactive element.

## key Features
- Responsive and modern UI design
- Dark and Light mode toggle
- Contact form with JavaScript validation
- Error and Sucess message handling
- hover effects and smooth animation

## Technologies Used
- HTML5
- CSS3
- JavaScript
